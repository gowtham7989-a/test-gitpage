Cloud-Native Architecture

Modern Pipeline: GLAPI & GKE Orchestration

Data is extracted from SQL Server via GLAPI calls and staged in Cloud Storage (Blob/GCS). Prefect, hosted on GKE, manages transformations and pushes data to Snowflake/MongoDB. Secrets are secured via HashiCorp Vault.

DevOps & Automation

Infrastructure Configuration: Ansible & Pester

Ansible playbooks automate SQL Server setup, while Pester validates the configuration state. The entire lifecycle is managed through a hybrid GitHub Actions and Azure DevOps (ADO) CI/CD pipeline.

Legacy Architecture

Legacy Pipeline: FTP & Task Scheduler

The traditional workflow moving data from SQL Server to FTP/NAS storage using Windows Task Scheduler, eventually feeding into Qlik Sense and PDW for business intelligence.
